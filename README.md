# Nodejs-2025

A Node.js utility project with random number generation functionality.

## Description

This project provides utility functions for Node.js applications, including random number generation capabilities.

## Features

- Random number generation (1-100)
- CommonJS module structure
- Reusable utility functions
- Reusable utility functions

## Installation

```bash
git clone https://github.com/kiganyamburu/Nodejs-2025.git
cd Nodejs-2025
npm install
```

## Usage

```javascript
const generateRandomNumber = require("./utils");

// Generate a random number between 1 and 100
const randomNum = generateRandomNumber();
console.log(randomNum);
```

## Project Structure

```
├── index.js          # Main entry point
├── utils.js          # Utility functions
├── package.json      # Project configuration
└── README.md         # Project documentation
```
